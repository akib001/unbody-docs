# Using SDK

### Setting Up Your Project


### 

The Unbody SDK allows developers to programmatically interact with the platform. Follow these steps to set up and make your first API call:

### **Step 1: Obtain Your API Key**

1. Log in to your Unbody dashboard.
2. Navigate to your project page and click the **Settings** tab.
3. Open the **Developer Settings** sub-page and create an API key.

### **Step 2: Initialize the SDK**

Install the Unbody SDK in your project and initialize it as follows:

```graphql
const unbody = new Unbody({
    apiKey: "<API_KEY>",
    projectId: "<PROJECT_ID>",
});
```

### **Step 3: Make Your First API Call**

Depending on your data source, here are some sample calls:

**Retrieve Google Docs:**

```graphql
const { data: { payload } } = await unbody.get.googleDoc.exec();
```

**Perform a Semantic Search:**

```graphql
const { data: { payload } } = await unbody.get
    .googleDoc
    .search
    .about("your query")
    .select("title", "autoSummary")
    .exec();
```

**Add Generative Features:**
You can group documents and apply a generative AI model:

```graphql
const { data: { generate } } = await unbody.get
    .googleDoc
    .search
    .about("context")
    .limit(2)
    .select("title", "autoSummary")
    .generate
    .fromMany(
        "your prompt",
        ["title", "autoSummary", "autoTopics"]
    )
    .exec();
```

### **Supported Sources**

Unbody SDK supports various sources, such as:

- Google Drive (Google Docs)
- Discord Messages
- GitHub Issues
- Text Blocks

Follow the same steps to retrieve, search, and generate data from these sources by substituting `googleDoc` with `discordMessage`, `githubComment`, or `textBlock` as applicable.
